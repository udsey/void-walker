**void-walker** is an autonomous agent that generates human-like personas to interact with [void-cast]([https://void-cast.fly.dev/](https://github.com/udsey/void-cast)).

Its primary job: test the site and seed it with realistic behavioral data. Each run produces a different persona (age, mood, country, language, archetype) that decides whether to enter, observes the space, and reflects on the experience.
